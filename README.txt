How to run locally:
1) Open index.html in a browser.
   (Or use a simple local server so links work consistently.)

Local server options:
- VS Code: Right-click index.html > Open with Live Server
- Python: python -m http.server 8000
  Then open: http://localhost:8000

Deploy options (static hosting):
- GitHub Pages: upload everything in this folder (root) and enable Pages.
- Netlify: drag-and-drop this folder.
- Vercel: deploy as "Other" / static files (no Next.js) by uploading this folder.
